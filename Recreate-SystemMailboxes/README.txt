Execution of this PowerShell Module is on your own risk. Please read the code before you run it. It will need a change of the execution policy for powershell.

The Filde should be saved in default PowerShell modules folder under a folder with the same name as the script.

If you run this script, you will recreate all exchange system mailboxes. 
It will assign the OAB capability to one of the system mailboxes and update OABs.
